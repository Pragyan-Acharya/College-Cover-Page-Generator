# College-Cover-Page-Generator 🎓📄
A fast, responsive, and free web app for College students to instantly generate, preview, and download professional assignment cover pages as A4 PDFs.

A fast, responsive, and completely free web application designed for College students across Nepal. This tool allows students to generate professional, print-ready assignment cover pages instantly without needing to sign up or use complex formatting software.

## ✨ Features

* **Real-Time Live Preview:** Watch your A4 cover page update instantly as you type in your details.
* **Native PDF Download/Print:** Uses the browser's highly reliable native print dialog to save sharp, perfect PDFs or print directly to paper without layout breaks.
* **Light & Dark Mode:** A beautiful, eye-care-friendly UI that seamlessly toggles between dark and light themes (saves your preference locally).
* **Smooth Mobile Experience:** Fully responsive layout with a custom slide-down hamburger menu and mobile-friendly horizontal scrolling for the live preview.
* **Custom Logo Upload:** Easily upload your specific college's logo to perfectly center on the cover page.
* **One-Click Reset:** Quickly clear all fields and return to the default CSIT template.
* **No Sign-up Required:** 100% free, client-side generation ensures your data stays private and fast.

## 🛠️ Tech Stack

This project is built purely with standard web technologies. No heavy frameworks or build tools are required!

* **HTML5:** Semantic structuring and form handling.
* **CSS3:** Custom styling, CSS Variables for theming, Flexbox/Grid for layout, and highly specific `@media print` queries for perfect A4 document rendering.
* **Vanilla JavaScript:** DOM manipulation, real-time input listening, and local storage management.
* **Boxicons:** For lightweight, crisp UI icons.
* **ScrollReveal.js:** For smooth, scroll-triggered fade-in animations.
